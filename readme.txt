镀膜车间用于检查镜片的镀膜效果的图形分析软件

主要作用是，根据下位机采集的数据，产生一个二维的坐标图。

原理简介：
  
    镜片经过镀膜工序之后，需要检查镀膜的效果，检测的原理是，通过指定波长的光照射到镀膜镜片上，镜片产生反射和透射的光，其强度的百分比的不同，再对比参考结果后，确定镀膜的效果。

指定波长的光，是单一波长的光，即指定频率的光。此光，是通过光栅单色仪产生指定波长的光，
再通过串口调节作用，使光栅单色仪产品连续的不同波长的光，做为镀膜检查用的光源，这样就能得到一个，透射光强度百分比与对应波长的二维数据信息，需要把此二维信息用二维坐标形式呈现出来，

数据信息需要分文件夹保存，备查。

上位机还要求控制光源的波长变化，包括指定固定波长值，设置步进波长的起点、终点、步进宽度。

相当于频谱仪的扫频宽度，


软件的名称应该是叫，光度计。资料来自网上， 参考：Lambda950操作指南，本软件参考其功能实现。
